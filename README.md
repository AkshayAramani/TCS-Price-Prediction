
# TCS Stock Price Prediction

## Objective
Predict TCS closing price using Linear Regression on live NSE data.

## Tools Used
Python, scikit-learn, yfinance, matplotlib, pandas

## Model
Simple Linear Regression with 80-20 train-test split

## Results
- R² Score: 0.954
- MSE: 2216
